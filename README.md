# AI-Powered Campaign Builder For Google Ads

Developed an end-to-end AI automation platform that compresses the Google Ads campaign creation workflow from ~3 days to under 60 seconds. I engineered the full-stack architecture and AI orchestration layer to transform a website or landing page into structured, ready-to-launch marketing campaigns.

## Key Technical Contributions:

- **AI Orchestration**: Designed an asynchronous pipeline using Python (FastAPI) and LangChain to handle multi-stage LLM tasks: web scraping, keyword extraction, and automated ad-group clustering.

- **Frontend Architecture**: Built a high-performance, responsive dashboard using TypeScript and Next.js, focusing on state management for real-time campaign previews.

- **API Integration**: Engineered a robust integration with the Google Ads API to programmatically publish complex campaign structures, including call-only and responsive search ads.

- **Data Integrity**: Implemented structured data parsing to ensure AI-generated ad copy and keywords adhered to strict Google Ads character limits and policy requirements.

- **Performance Scaling**: Deployed the serverless architecture using Docker and Google Cloud Run, ensuring the system scales automatically to handle concurrent AI processing requests.

## Tech Stack: 
TypeScript, Next.js, Python, FastAPI, LangChain (LLMs), Docker, Google Cloud Run, Google Ads API.
